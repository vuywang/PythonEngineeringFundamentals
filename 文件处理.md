# 1. 文件处理到底在处理什么？🤔

你可以把文件理解成电脑硬盘上的一块数据。

比如：

```text
hello.txt
data.csv
config.json
logo.png
student.txt
```

Python 可以对这些文件做很多事：

* 读取文件内容 📖
* 写入新内容 ✍️
* 追加内容 ➕
* 逐行读取 🔍
* 读取 CSV 表格 📊
* 读取 JSON 配置 🧾
* 读取图片、音频这种二进制文件 🖼️

---

# 2. Python 文件处理的基本格式 🧱

最基础写法是：

```python
file = open("test.txt", "r", encoding="utf-8")
content = file.read()
print(content)
file.close()
```

这段代码意思是：

```python
open("test.txt", "r", encoding="utf-8")
```

打开 `test.txt` 文件。

```python
file.read()
```

读取文件内容。

```python
file.close()
```

关闭文件。

---

# 3. 为什么一定要关闭文件？🔒

文件打开后，会占用系统资源。

如果你一直不关闭，可能会导致：

* 文件资源没释放
* 文件内容没完全写入
* Windows 上文件被占用
* 程序运行时间长了出问题

所以以前常见写法是：

```python
file = open("test.txt", "r", encoding="utf-8")
try:
    content = file.read()
    print(content)
finally:
    file.close()
```

但是这种写法有点麻烦。

所以 Python 更推荐用 `with open(...)`。

---

# 4. 最推荐写法：`with open(...)` ⭐

```python
with open("test.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
```

这就是你以后最常用的写法 ✅

它的好处是：

> **代码执行完后，会自动关闭文件**

你不用手动写：

```python
file.close()
```

以后你写文件处理，优先写这种。

---

# 5. `open()` 的三个重要参数 🧩

最常用的是这三个：

```python
open("文件路径", "打开模式", encoding="编码")
```

比如：

```python
open("test.txt", "r", encoding="utf-8")
```

## 第一个参数：文件路径 📍

```python
"test.txt"
```

表示当前项目目录下的 `test.txt`。

也可以写：

```python
"data/student.txt"
```

表示 `data` 文件夹里的 `student.txt`。

---

## 第二个参数：打开模式 🧭

比如：

```python
"r"
"w"
"a"
"rb"
"wb"
```

这个决定你是读取、写入、追加，还是处理二进制文件。Python 官方文档列出了常见模式：`'r'` 表示读取，`'w'` 表示写入并先清空原文件，`'x'` 表示独占创建，`'a'` 表示追加，`'b'` 表示二进制模式，`'t'` 表示文本模式，`'+'` 表示读写更新。([Python documentation][1])

---

## 第三个参数：编码方式 🌐

```python
encoding="utf-8"
```

建议你处理文本文件时，基本都写上它。

尤其是中文文件。

比如：

```python
with open("hello.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

如果不写编码，文本模式下默认编码会依赖当前平台环境，容易出现不同电脑表现不一致的问题。([Python documentation][1])

---

# 6. 文件打开模式详解 📚

## 6.1 `r`：读取文件 📖

```python
with open("test.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
```

注意：

如果文件不存在，会报错：

```python
FileNotFoundError
```

比如：

```python
with open("not_exists.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

因为你要读的文件根本没有。

---

## 6.2 `w`：写入文件 ✍️

```python
with open("test.txt", "w", encoding="utf-8") as file:
    file.write("你好，Python 文件处理！")
```

如果 `test.txt` 不存在，会自动创建。

但是注意 ⚠️

> **`w` 会清空原来的内容，再写入新内容。**

比如原来文件里有：

```text
旧内容
```

你执行：

```python
with open("test.txt", "w", encoding="utf-8") as file:
    file.write("新内容")
```

文件最后只剩：

```text
新内容
```

旧内容没了 😵

---

## 6.3 `a`：追加写入 ➕

```python
with open("test.txt", "a", encoding="utf-8") as file:
    file.write("这是追加内容\n")
```

`a` 不会清空原内容，而是在文件末尾继续加内容。

比如原来：

```text
第一行
```

执行追加：

```python
with open("test.txt", "a", encoding="utf-8") as file:
    file.write("第二行\n")
```

结果：

```text
第一行
第二行
```

---

## 6.4 `x`：创建新文件 🆕

```python
with open("new_file.txt", "x", encoding="utf-8") as file:
    file.write("新文件内容")
```

`x` 的意思是：

> 只允许创建新文件，如果文件已经存在，就报错

适合防止不小心覆盖旧文件。

---

## 6.5 `b`：二进制模式 🖼️

处理图片、视频、压缩包时，要用二进制模式。

读取图片：

```python
with open("logo.png", "rb") as file:
    data = file.read()
```

写入图片副本：

```python
with open("logo.png", "rb") as source:
    data = source.read()

with open("logo_copy.png", "wb") as target:
    target.write(data)
```

文本模式读写的是 `str`，二进制模式读写的是 `bytes`；Python 的 `io` 文档也明确区分了文本 I/O 和二进制 I/O，文本 I/O 处理 `str`，二进制 I/O 处理 `bytes`。([Python documentation][2])

# 7. `read()`：一次性读取全部内容 📖

```python
with open("test.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
```

如果文件内容是：

```text
第一行
第二行
第三行
```

输出就是：

```text
第一行
第二行
第三行
```

这个方式很简单，但有一个问题：

> 如果文件特别大，一次性全部读进内存可能不合适。

小文件没问题，大文件要逐行读。

---

# 8. `readline()`：一次读取一行 🧵

```python
with open("test.txt", "r", encoding="utf-8") as file:
    line1 = file.readline()
    line2 = file.readline()

    print(line1)
    print(line2)
```

它每次读取一行。

比如文件内容是：

```text
第一行
第二行
第三行
```

第一次 `readline()` 读到：

```text
第一行
```

第二次读到：

```text
第二行
```

---

# 9. `readlines()`：读取所有行，变成列表 📋

```python
with open("test.txt", "r", encoding="utf-8") as file:
    lines = file.readlines()
    print(lines)
```

如果文件内容是：

```text
第一行
第二行
第三行
```

结果类似：

```python
['第一行\n', '第二行\n', '第三行\n']
```

注意每一行后面可能带着 `\n`。

---

# 10. 最推荐的大文件读取方式：for 循环逐行读 🔥

```python
with open("test.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())
```

这里：

```python
for line in file:
```

表示一行一行读取。

```python
line.strip()
```

去掉每行两边的空白字符，包括换行符 `\n`。

这种写法适合读取日志、CSV、TXT 大文件。

---

# 11. `write()`：写入字符串 ✍️

```python
with open("test.txt", "w", encoding="utf-8") as file:
    file.write("第一行\n")
    file.write("第二行\n")
```

文件内容会变成：

```text
第一行
第二行
```

注意，`write()` 不会自动换行。
你要自己写：

```python
\n
```

---

# 12. `writelines()`：写入多行 📋

```python
lines = ["第一行\n", "第二行\n", "第三行\n"]

with open("test.txt", "w", encoding="utf-8") as file:
    file.writelines(lines)
```

注意：

`writelines()` 也不会自动帮你加换行。
所以列表里的字符串要自己带 `\n`。

# 13. 换行符问题：`\n`、CRLF、LF 🌊

你前面刚问过 CRLF 和 LF，这里刚好接上。

在 Python 文本模式中，换行会涉及自动转换。官方文档说明：读取时如果 `newline=None`，会启用通用换行模式，输入中的 `\n`、`\r`、`\r\n` 会被转换为 `\n`；写入时如果 `newline=None`，写入的 `\n` 会被转换成系统默认换行符。([Python documentation][1])

所以如果你想尽量稳定地写出 LF，可以这样：

```python
with open("test.txt", "w", encoding="utf-8", newline="\n") as file:
    file.write("第一行\n")
    file.write("第二行\n")
```

但是一般写普通文本，你不用每次都写 `newline="\n"`。
更常见的是：

```python
with open("test.txt", "w", encoding="utf-8") as file:
    file.write("第一行\n")
```

# 14. 文件路径怎么写？📍

文件路径有两种：

## 14.1 相对路径

```python
with open("data.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

意思是从当前运行目录找 `data.txt`。

也可以：

```python
with open("data/student.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

意思是找 `data` 文件夹里的 `student.txt`。

---

## 14.2 绝对路径

Windows 示例：

```python
with open("C:/Users/lili/Desktop/data.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

也可以写成原始字符串：

```python
with open(r"C:\Users\lili\Desktop\data.txt", "r", encoding="utf-8") as file:
    print(file.read())
```

注意这里的：

```python
r"..."
```

表示原始字符串，可以减少反斜杠转义问题。

---

# 15. 更推荐的路径写法：`pathlib` 📦

现代 Python 里，处理路径更推荐用 `pathlib`。

```python
from pathlib import Path

path = Path("data") / "student.txt"

with path.open("r", encoding="utf-8") as file:
    print(file.read())
```

`pathlib` 是 Python 标准库中面向对象的文件路径模块，它提供了适合不同操作系统语义的路径类；官方文档也说，如果不确定用哪个类，通常使用 `Path` 就够了。([Python documentation][3])

# 22. 读写 JSON 文件 🧾

JSON 很常用于配置文件、接口数据、前后端交互。

Python 标准库的 `json` 模块可以把 Python 数据结构转成 JSON 字符串，也可以把 JSON 字符串还原成 Python 数据结构；官方教程把这两个过程叫做序列化和反序列化。([Python documentation][4])

---

## 22.1 写 JSON 文件

```python
import json

student = {
    "name": "张三",
    "age": 18,
    "score": 95
}

with open("student.json", "w", encoding="utf-8") as file:
    json.dump(student, file, ensure_ascii=False, indent=4)
```

这里：

```python
ensure_ascii=False
```

表示中文不要转成 Unicode 转义。

```python
indent=4
```

表示格式化缩进，看起来更舒服。

生成的文件内容类似：

```json
{
    "name": "张三",
    "age": 18,
    "score": 95
}
```

---

## 22.2 读 JSON 文件

```python
import json

with open("student.json", "r", encoding="utf-8") as file:
    student = json.load(file)

print(student)
print(student["name"])
```

输出：

```python
{'name': '张三', 'age': 18, 'score': 95}
张三
```

# 23. `json.dump()` 和 `json.dumps()` 的区别 🧠

这个很容易混。

## `json.dump()`

直接写入文件：

```python
json.dump(data, file)
```

## `json.dumps()`

转成字符串：

```python
json_str = json.dumps(data, ensure_ascii=False)
```

例子：

```python
import json

data = {"name": "张三"}

json_str = json.dumps(data, ensure_ascii=False)
print(json_str)
```

输出：

```json
{"name": "张三"}
```

官方文档的示例也展示了 `json.dumps()` 返回字符串，`json.dump()` 可以把对象写入类似文件的流对象。([Python documentation][5])

# 24. 读写 CSV 文件 📊

CSV 很像表格，常用于数据分析。

比如 `students.csv`：

```csv
name,age,score
张三,18,95
李四,19,88
王五,20,76
```

Python 标准库 `csv` 模块用于读取和写入 CSV 表格数据，并提供 `reader`、`writer`、`DictReader`、`DictWriter` 等对象。([Python documentation][6])

---

## 24.1 读取 CSV：普通 reader

```python
import csv

with open("students.csv", "r", encoding="utf-8", newline="") as file:
    reader = csv.reader(file)

    for row in reader:
        print(row)
```

输出类似：

```python
['name', 'age', 'score']
['张三', '18', '95']
['李四', '19', '88']
['王五', '20', '76']
```

注意：

CSV 官方文档建议，如果传入的是文件对象，打开时应使用 `newline=''`。([Python documentation][6])

---

## 24.2 读取 CSV：DictReader 更好用 ✅

```python
import csv

with open("students.csv", "r", encoding="utf-8", newline="") as file:
    reader = csv.DictReader(file)

    for row in reader:
        print(row["name"], row["age"], row["score"])
```

输出：

```text
张三 18 95
李四 19 88
王五 20 76
```

这种方式更像按字段名取值，比下标更清楚。

---

## 24.3 写入 CSV：writer

```python
import csv

rows = [
    ["name", "age", "score"],
    ["张三", 18, 95],
    ["李四", 19, 88],
    ["王五", 20, 76]
]

with open("students.csv", "w", encoding="utf-8", newline="") as file:
    writer = csv.writer(file)
    writer.writerows(rows)
```

---

## 24.4 写入 CSV：DictWriter

```python
import csv

students = [
    {"name": "张三", "age": 18, "score": 95},
    {"name": "李四", "age": 19, "score": 88},
    {"name": "王五", "age": 20, "score": 76}
]

with open("students.csv", "w", encoding="utf-8", newline="") as file:
    fieldnames = ["name", "age", "score"]
    writer = csv.DictWriter(file, fieldnames=fieldnames)

    writer.writeheader()
    writer.writerows(students)
```

这个更适合处理字典列表。

# 25. 文件复制案例 🧪

复制文本文件：

```python
with open("source.txt", "r", encoding="utf-8") as source:
    content = source.read()

with open("target.txt", "w", encoding="utf-8") as target:
    target.write(content)
```

复制二进制文件，比如图片：

```python
with open("source.png", "rb") as source:
    data = source.read()

with open("target.png", "wb") as target:
    target.write(data)
```

如果文件很大，更推荐分块复制：

```python
with open("source.zip", "rb") as source:
    with open("target.zip", "wb") as target:
        while True:
            chunk = source.read(1024 * 1024)
            if not chunk:
                break
            target.write(chunk)
```

这里每次读取 1MB，避免一次性把大文件全部读进内存。

# 26. 实战：统计文本文件行数、字数、字符数 📊

假设 `article.txt` 是文章。

```python
line_count = 0
word_count = 0
char_count = 0

with open("article.txt", "r", encoding="utf-8") as file:
    for line in file:
        line_count += 1
        char_count += len(line)
        word_count += len(line.split())

print("行数：", line_count)
print("单词数：", word_count)
print("字符数：", char_count)
```

中文文章的“词数”不能简单用 `split()` 准确统计，但英文文本可以先这么练。

# 28. 实战：读取学生成绩并计算平均分 🎓

CSV 文件 `students.csv`：

```csv
name,age,score
张三,18,95
李四,19,88
王五,20,76
```

代码：

```python
import csv

total = 0
count = 0

with open("students.csv", "r", encoding="utf-8", newline="") as file:
    reader = csv.DictReader(file)

    for row in reader:
        score = int(row["score"])
        total += score
        count += 1

avg = total / count
print("平均分：", avg)
```

输出：

```text
平均分： 86.33333333333333
```

格式化一下：

```python
print(f"平均分：{avg:.2f}")
```

输出：

```text
平均分：86.33
```

# 30. 文件处理和异常结合：更稳的写法 🛡️

```python
from pathlib import Path

path = Path("data.txt")

try:
    content = path.read_text(encoding="utf-8")
except FileNotFoundError:
    print("文件不存在")
except UnicodeDecodeError:
    print("文件编码不正确")
else:
    print(content)
```

这里处理了两个常见问题：

* 文件不存在
* 编码不对

# 31. 你最容易踩的坑 🚨

## 坑1：忘记写 `encoding="utf-8"`

```python
open("test.txt", "r")
```

中文文件可能出问题。

更推荐：

```python
open("test.txt", "r", encoding="utf-8")
```

---

## 坑2：用 `w` 不小心清空文件

```python
open("test.txt", "w", encoding="utf-8")
```

`w` 会覆盖旧内容。
不想覆盖，用：

```python
"a"
```

---

## 坑3：写入时忘记加 `\n`

```python
file.write("第一行")
file.write("第二行")
```

结果会变成：

```text
第一行第二行
```

应该：

```python
file.write("第一行\n")
file.write("第二行\n")
```

---

## 坑4：路径找不到

你以为文件在桌面，但程序其实从项目目录找。

可以打印当前工作目录：

```python
from pathlib import Path

print(Path.cwd())
```

看看 Python 到底从哪里找文件。

---

## 坑5：大文件一次性 `read()`

```python
content = file.read()
```

小文件可以，大文件别乱用。
大文件优先：

```python
for line in file:
    ...
```

# 36. 最后总结一下 🌟

```python
with open("文件路径", "模式", encoding="utf-8") as file:
    内容操作
```

最常用模式：

```python
r   读取
w   写入，会覆盖
a   追加
rb  二进制读取
wb  二进制写入
```

最常用读写：

```python
read()
readline()
readlines()
write()
writelines()
```

最推荐习惯：

```python
encoding="utf-8"
with open(...)
pathlib.Path
异常处理
```
