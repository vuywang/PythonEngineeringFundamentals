# 1. 什么是函数？🤔

函数（function），你可以先把它理解成：

> **一段可以反复使用的代码块**

比如你要多次做“两个数相加”这件事，如果每次都手写：

```python
a = 10
b = 20
print(a + b)
```

那很麻烦。
这时候就可以把“相加”这个动作封装成一个函数，后面想用的时候直接调用就行 ✅

---

# 2. 为什么要用函数？📌

函数的作用非常大，主要有这几个：

### （1）减少重复代码 🔁

同样的逻辑，不要写很多遍。

### （2）让代码更清晰 👀

看到函数名，就大概知道它是干什么的。

### （3）方便维护 🛠️

以后要改逻辑，只改函数内部一处就行。

### （4）方便复用 ♻️

一个函数可以在很多地方调用。

---

# 3. 最基本的函数长什么样？🧱

Python 定义函数用 `def`

格式：

```python
def 函数名():
    函数体
```

例子：

```python
def say_hello():
    print("你好，欢迎学习 Python 函数！")
```

这里：

* `def`：表示定义函数
* `say_hello`：函数名
* `()`：参数位置
* `:`：不能少
* 缩进的内容：函数体

---

# 4. 函数定义了，不代表执行了 ⚠️

很多初学者会犯这个错。

看这个：

```python
def say_hello():
    print("你好，欢迎学习 Python 函数！")
```

这只是 **定义** 了函数，不会自动执行。

要执行，得 **调用函数**：

```python
say_hello()
```

完整写法：

```python
def say_hello():
    print("你好，欢迎学习 Python 函数！")

say_hello()
```

输出：

```python
你好，欢迎学习 Python 函数！
```

---

# 5. 函数可以反复调用 🔄

```python
def say_hello():
    print("你好！")

say_hello()
say_hello()
say_hello()
```

输出：

```python
你好！
你好！
你好！
```

这就是函数最大的意义之一：**写一次，用很多次** 🎉

# 6. 带参数的函数 🧩

很多时候，函数不是固定干一件死事，而是希望“传不同的数据，做不同的处理”。

这时候就要用 **参数**。

## 6.1 一个参数

```python
def greet(name):
    print("你好，" + name)
```

调用：

```python
greet("小明")
greet("小红")
```

输出：

```python
你好，小明
你好，小红
```

这里的 `name` 就是参数。

你可以理解成：
函数像一个机器，`name` 是你塞进去的原料 🏭

---

## 6.2 多个参数

```python
def add(a, b):
    print(a + b)
```

调用：

```python
add(3, 5)
add(10, 20)
```

输出：

```python
8
30
```

这里 `a` 和 `b` 都是参数。

---

# 7. 参数和实参的区别 🧠

这个概念很容易混。

看例子：

```python
def add(a, b):
    print(a + b)

add(3, 5)
```

这里：

* `a`、`b`：**形参**（形式参数）
  是函数定义时写的名字
* `3`、`5`：**实参**（实际参数）
  是调用函数时真正传进去的值

你可以简单记成：

* 定义时写的是“形参”
* 调用时给的是“实参”

# 8. return 是什么？⭐

这是函数最核心的知识之一。

很多新手会把 `print()` 和 `return` 搞混。

## 8.1 `print()` 只是打印

```python
def add(a, b):
    print(a + b)

result = add(3, 5)
print("result =", result)
```

输出：

```python
8
result = None
```

为什么 `result` 是 `None`？

因为这个函数只是打印了结果，但 **没有返回结果**。

---

## 8.2 `return` 才是返回值

```python
def add(a, b):
    return a + b

result = add(3, 5)
print("result =", result)
```

输出：

```python
result = 8
```

### 重点理解：

* `print()`：给你看
* `return`：把结果交出去

这个区别非常重要 ⚠️

---

# 9. return 的执行特点 🚪

一旦执行到 `return`，函数就结束了。

```python
def test():
    print("开始")
    return 100
    print("结束")
```

调用：

```python
print(test())
```

输出：

```python
开始
100
```

`return` 后面的 `print("结束")` 根本不会执行。

# 10. 一个函数可以返回多个值吗？🎁

可以。

```python
def get_user():
    name = "张三"
    age = 20
    return name, age
```

调用：

```python
result = get_user()
print(result)
```

输出：

```python
('张三', 20)
```

其实返回的是一个元组。

也可以拆开接收：

```python
name, age = get_user()
print(name)
print(age)
```

输出：

```python
张三
20
```

# 11. 默认参数 🪄

有时候某个参数通常都是固定值，那就可以给它一个默认值。

```python
def greet(name, msg="你好"):
    print(msg, name)
```

调用：

```python
greet("小明")
greet("小红", "欢迎你")
```

输出：

```python
你好 小明
欢迎你 小红
```

解释：

* 没传 `msg` 时，用默认值 `"你好"`
* 传了，就用你传的

# 12. 位置参数和关键字参数 📍

## 12.1 位置参数

调用时按顺序传：

```python
def introduce(name, age):
    print("姓名：", name)
    print("年龄：", age)

introduce("小明", 18)
```

这里 `"小明"` 给 `name`，`18` 给 `age`

---

## 12.2 关键字参数

调用时明确写出参数名：

```python
introduce(age=18, name="小明")
```

顺序可以变，因为你已经写清楚了。

---

## 12.3 混合使用

```python
introduce("小明", age=18)
```

可以。
但要注意：

> **位置参数必须放在关键字参数前面**

下面这样会报错 ❌

```python
introduce(name="小明", 18)
```

# 13. 可变参数 `*args` 🌈

有时候你不知道会传多少个参数，这时可以用 `*args`

```python
def add_all(*args):
    print(args)
```

调用：

```python
add_all(1, 2, 3)
add_all(10, 20, 30, 40)
```

输出：

```python
(1, 2, 3)
(10, 20, 30, 40)
```

说明：
`*args` 会把多个位置参数收集成一个 **元组**

---

## 13.1 实际用途

```python
def add_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(add_all(1, 2, 3))
print(add_all(10, 20, 30, 40))
```

输出：

```python
6
100
```

# 14. 可变关键字参数 `**kwargs` 🎒

如果你想接收不确定数量的“键=值”参数，就用 `**kwargs`

```python
def show_info(**kwargs):
    print(kwargs)
```

调用：

```python
show_info(name="小明", age=18, city="北京")
```

输出：

```python
{'name': '小明', 'age': 18, 'city': '北京'}
```

说明：

* `**kwargs` 会把多个关键字参数收集成一个 **字典**

---

## 14.1 实际用途

```python
def show_info(**kwargs):
    for key, value in kwargs.items():
        print(key, ":", value)

show_info(name="小明", age=18, city="北京")
```

输出：

```python
name : 小明
age : 18
city : 北京
```

# 16. 局部变量和全局变量 🌍

这也是函数里非常重要的一块。

## 16.1 局部变量

在函数内部定义的变量，只能在函数内部使用。

```python
def test():
    x = 10
    print(x)

test()
```

输出：

```python
10
```

但你在函数外面用 `x` 就不行：

```python
def test():
    x = 10

test()
print(x)
```

会报错，因为 `x` 是局部变量。

---

## 16.2 全局变量

在函数外定义的变量，叫全局变量。

```python
x = 100

def test():
    print(x)

test()
```

输出：

```python
100
```

函数内部可以读取全局变量。

---

## 16.3 修改全局变量要小心

```python
x = 100

def test():
    x = 200
    print("函数内部：", x)

test()
print("函数外部：", x)
```

输出：

```python
函数内部： 200
函数外部： 100
```

这里函数里的 `x` 和外面的 `x` 不是同一个。

如果你真的要在函数里修改全局变量，要用 `global`

```python
x = 100

def test():
    global x
    x = 200

test()
print(x)
```

输出：

```python
200
```

⚠️ 不过初学阶段不建议乱用 `global`，容易让代码变乱。


输出：

```python

```


这里没有加括号，表示把函数本身给变量。

# 20. lambda 匿名函数 ⚡

有时候函数非常简单，只想临时用一下，可以写成 `lambda`

普通写法：

```python
def add(a, b):
    return a + b
```

lambda 写法：

```python
add = lambda a, b: a + b
print(add(3, 5))
```

输出：

```python
8
```

### lambda 特点：

* 适合简单函数
* 一般只写一行
* 不适合复杂逻辑

常见场景：排序、映射、过滤

例如排序：

```python
students = [
    {"name": "小明", "age": 20},
    {"name": "小红", "age": 18},
    {"name": "小刚", "age": 22}
]

students.sort(key=lambda x: x["age"])
print(students)
```

这就是匿名函数的典型用法。

# 24. 函数设计时常见误区 🚨

## 24.1 只会 print，不会 return

错误思路：

```python
def add(a, b):
    print(a + b)
```

这样函数复用性差。

更好的写法：

```python
def add(a, b):
    return a + b
```

然后你想打印就自己决定：

```python
result = add(3, 5)
print(result)
```

---

## 24.2 一个函数干太多事

不推荐这种：

```python
def handle_user():
    # 读取文件
    # 登录验证
    # 写数据库
    # 发邮件
    # 生成报表
    pass
```

函数太大，很难维护 😵

更好的方式是拆开：

```python
def read_file():
    pass

def check_login():
    pass

def save_data():
    pass
```

---

## 24.3 参数太混乱

比如：

```python
def test(a, b, c, d, e, f, g):
    pass
```

这种函数不利于阅读。
参数太多时，要考虑是否该拆分逻辑。

# 26. 递归函数是什么？🌀

递归就是：

> **函数自己调用自己**

比如计算 `5 + 4 + 3 + 2 + 1`

```python
def sum_n(n):
    if n == 1:
        return 1
    return n + sum_n(n - 1)

print(sum_n(5))
```

输出：

```python
15
```

递归有两个关键点：

### （1）必须有结束条件

不然会无限调用。

### （2）每次调用都要向结束条件靠近

比如上面 `n - 1`，就是一步步靠近 `1`

⚠️ 初学阶段知道递归概念就行，不一定马上用得多。

# 27. 函数执行过程到底发生了什么？🔍

看这个：

```python
def add(a, b):
    result = a + b
    return result

x = add(3, 5)
print(x)
```

执行过程可以理解成：

### 第一步

Python 看到 `def add(a, b):`
把这个函数记住。

### 第二步

执行 `x = add(3, 5)`
函数被调用，`a=3`，`b=5`

### 第三步

执行函数内部：

```python
result = a + b
```

得到 `8`

### 第四步

执行：

```python
return result
```

把 `8` 返回给调用处

### 第五步

于是：

```python
x = 8
```

最后打印 `x`

# 30. 一个完整的小案例 🌟

# 34. 最后帮你做一个总总结 🎯

Python 函数你可以先这样理解：

### 核心结构

```python
def 函数名(参数):
    代码
    return 返回值
```

### 最关键的三件事

* **函数名**：这个功能叫什么
* **参数**：需要什么输入
* **返回值**：最后给出什么结果

### 最容易错的地方

* 定义了函数却没调用
* 只写 `print` 不写 `return`
* 参数顺序弄错
